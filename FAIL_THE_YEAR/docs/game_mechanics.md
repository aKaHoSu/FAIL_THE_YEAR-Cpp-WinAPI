# FAIL THE YEAR - ゲームメカニクス仕様

> このドキュメントは、ゲーム概要・ルール・オブジェクト仕様をまとめたものです。  
> シーン遷移の詳細は `game_specification.md` / `game_flowchart.md` を参照してください。

---

## 1. ゲーム概要

### コンセプト
落下してくる「留年」オブジェクトを回避し、棒人間くんを進級させる落下物回避ゲーム。

### 基本情報

| 項目 | 値 |
|---|---|
| ウィンドウサイズ | 1200 × 700 px |
| 地面 Y 座標（`FIX`） | 597 px |
| 右端有効 X 座標（`RIGHT_FIX`） | 1126 px |
| タイマー更新 | 10ms |

---

## 2. ゲームルール サマリー

### 通常
- 難易度は、 "Easy" "Hard" "Nightmare" の三段階。難易度によって初期ライフや回避目標数が異なる。:

| 難易度 | ライフ | 回避目標数（`evadeGoal`） | ボムCT（フレーム） | 
|---|---|---|---|
| Easy | 5 | 150 | 3000 |
| Hard | 3 | 450 | 3000 |
| Nightmare | 1 | 700 | 2000 |

- 画面上部から「留年」が落下してくるので、プレイヤーはそれを回避する。

### クリア（進級）条件
- `evadeCnt >= evadeGoal` で **進級確定**（`canAdv = true`）。
- 落ちてくる「留年(黒色文字)」が「進級(緑色文字)」に変化。
- 進級確定後はライフが 1 にリセットされ、「進級」に当たるとゲームクリア。リザルトへ遷移。
- 「進級」を避けた数は、"追い込み数"としてカウントされる。

### 敗北条件（留年確定）
- 進級確定していない状態でライフが 0 になると **留年確定**。リザルトへ遷移。

### デバッグモード（Debug構成のときのみ有効）
- `/` キーでコマンドラインを表示。既定のコマンドを入力することで、以下の効果を発動。:

| コマンド | 効果 |
|---|---|
| `>` godmode | ヒットチェック無効化 |
| `>` hitbox | collisionを持つオブジェクトのヒットボックスを描画 |
| `>` firstplay | 初回プレイフラグをfalse |
| `>` clearflag | クリアフラグをtrue |
| `>` clearnmflag | ナイトメアクリアフラグをtrue |
| `>` escnmflag | ナイトメア脱出済みフラグをtrue |
| `>` help | 有効なコマンドの一覧を表示 |

- コマンドはトグル式で、再度入力することで効果を解除できる（`help` を除く）。

## 3. プレイヤー仕様

### 概要
- 十字キーで移動。
- `Shift` を押している間は低速移動。
- `Z` でボム発動。
- 残りのライフが 1 のときは震え演出。
- 留年オブジェクトに当たるとダメージを受け、一定時間無敵状態になる（点滅演出）。
- `Landed` 状態の留年・進級オブジェクトには当たらない
- ライフが 0 になるとゲーム終了。

### 基本パラメータ

| 項目 | 値 | 定数名 |
|---|---|---|
| 描画サイズ | 56 × 62 px | `PLAYER_WIDTH` / `PLAYER_HEIGHT` |
| ヒットボックス | 40 × 46 px | `collision` |
| 通常移動速度 | 8 px/frame | `kSpeedNormal` |
| 低速移動速度（Shift） | 4 px/frame | `kSpeedSlow` |
| ダメージ無敵時間 | 100 フレーム | `DMG_CT` |

### 移動範囲

| 方向 | 範囲 |
|---|---|
| X | 0 ～ `RIGHT_FIX`（1126）px |
| Y | 0 ～ `FIX`（597）px |

### プレイヤー状態

| 状態（`PlayerStatus`） | 説明 |
|---|---|
| `None` | 無効（`Ready` シーン以前） |
| `Easy` | 通常 |
| `Damage` | ダメージ受け中。`DMG_CT(100)` フレームの間点滅し、終了後 `Easy` に戻る |

### ボム

- `hasBomb == true` のとき `Z` キーで発動可能。
- ボムが使用可能なときは足元に魔法陣が表示される。
- 発動後は `hasBomb = false` になり、`bombCT` フレーム経過後に再度使用可能になる。

## 4. 留年・進級オブジェクト仕様

### 概要
- 画面上部から落下してくるオブジェクト。
- 進級確定前は「留年(黒色文字)」、確定後は「進級(緑色文字)」として描画。
- 落下中は重力加速で Y 座標が増加し、地面(`FIX`)に着地するとバウンドして跳ねる。
- 着地後は `Landed` 状態となり、灰色で描画される。

### 基本パラメータ

| 項目 | 値 | 定数名 |
|---|---|---|
| 描画サイズ / ヒットボックス | 41 × 80 px | `RYUNEN_WIDTH` / `RYUNEN_HEIGHT` |
| 重力加速度 | 0.15 | `gravity` |
| 落下量（毎フレーム） | `gravity × actCnt` px | — |
| 着地判定 Y | `FIX + 60` = 657 px | `kFixTolerance` |
| 場外判定 Y | `WINDOW_H + 20` = 720 px | `kOffscreenMargin` |
| 表示フォント | @メイリオ 30pt・90度回転 | `kFontSize` |

### 状態遷移

```
None ──→ Falling ──→ Landed ──→ Offscreen
```

| 状態（`RyunenStatus`） | 説明 |
|---|---|
| `None` | 無効（`Playing` シーン以前） |
| `Falling` | 落下中。重力加速で Y が増加 |
| `Landed` | 着地。バウンド移動を行う。ヒットチェックが発生しない |
| `Offscreen` | 画面外。 

### 描画色

| 条件 | 色 |
|---|---|
| 落下中（Easy / Hard・進級未確定） | 黒（RGB(30, 30, 30)） |
| 落下中（Nightmare） | 赤（RGB(100, 0, 0)） |
| 進級確定後（`canAdv == true`）・「進級」表示 | 緑（RGB(0, 140, 0)） |
| 着地（`Landed`） | 灰色（RGB(160, 160, 160)） |
| ゴッドモード有効 | マゼンタ（RGB(255, 0, 255)） |

### スポーン仕様

| 難易度 | スポーン間隔（通常） | スポーン間隔（進級確定後） | 同時出現上限 |
|---|---|---|---|
| Easy | 100 フレーム | 50 フレーム | 27 体（`MAX_RYUNEN - 45`） |
| Hard | 70 フレーム | 50 フレーム | 52 体（`MAX_RYUNEN - 20`） |
| Nightmare | 50 フレーム | 50 フレーム | 72 体（`MAX_RYUNEN`） |

#### スポーン X 座標

0 番目と 1 番目の留年は、画面の左右端付近に固定でスポーンする。

| インデックス | スポーン位置 |
|---|---|
| 0 番目 | x = 50（固定・左端付近） |
| 1 番目 | x = 1190（固定・右端付近） |
| 2 番目以降 | x = ランダム（100 ～ 1126） |

## 8. チュートリアル

### 概要
- メニューから選択可能。
- 8 つのステップを順に進む形式。
- `Enter` で次のステップへ、`Esc` で前のステップへ戻る。
- 最初のステップ（`Consept`）で `Esc` を押すとメニューへ戻る。
- 最後のステップ（`EndTutorial`）で `Enter` を押すとタイトルへ戻る。
- チュートリアル内の操作は、ゲーム本編には影響しない。

### ステップ一覧

| # | ステップ（`TutorialStepId`） | 概要 |
|---|---|---|
| 1 | `Consept` | ゲームのコンセプト説明。 |
| 2 | `Move` | 移動操作の説明。プレイヤー操作可能。 |
| 3 | `RyunenObject` | 留年オブジェクトの説明。プレイヤーが留年オブジェクトに当たるとどうなるのかを明確に説明する。 |
| 4 | `Life` | SUN値（ライフ）の説明。留年オブジェクトに被弾するとライフが 1 に固定される(震え演出の明示)。 |
| 5 | `Bomb` | ボムの説明。留年 5 体がスポーン。ボムCT が 100 フレームに短縮 |
| 6 | `ClearCondition` | クリア条件の説明。 |
| 7 | `AfterCanAdv` | 進級確定後の挙動説明。`canAdv = true` 状態にして「進級」オブジェクトを 1 体スポーンさせる。 |
| 8 | `EndTutorial` | チュートリアル終了メッセージ |

### ステップ別の特記事項

**`RyunenObject`**、**`Life`**
- 留年オブジェクトのx軸・プレイヤーの位置は固定。
- 留年オブジェクトに一度当たった後から、プレイヤーは操作可能になる。

**`Bomb`**
- 留年 5 体が x = 900 / 950 / 1000 / 1050 / 1100 に順に固定スポーン。
- ボムCT が 100 フレームに短縮されており、繰り返し発動して確認できるようにしている。

## 9. Nightmare モード

Nightmareモードは隠し要素。非常に高難易度なモードです。  

### 解放条件

- Easy または Hard をクリアすると `hasEverCleared = true` になり、難易度選択画面に見切れた "N" の選択肢が出現、選択可能になる。

### AfterResult **（Nightmare クリア時）** 

- `hasEverClearedNightmare = true` に設定される。
- 次回以降の Nightmare AfterResult では通常の Retry / Title 選択肢が表示される。

### AfterResult **（Nightmare 留年確定時）** 

通常の Retry / Title 選択肢は表示されず、専用のテキスト・演出に切り替わる。

**`Enter` キー**
- カーソル位置に関わらずリトライ（`Ready` シーンへ遷移）。

**Esc 連打**
- `Esc` を 1 回押すごとに `escCnt` が 1 増加する。
- `Esc` を押さない時間が続くと `escCnt` が自動で減少する（7 フレームの遅延後、10 フレームごとに 1 ずつ減少）。
- `escCnt >= 30` に達すると `EscNightmare` シーンへ遷移し、難易度が Easy にリセットされる。

**テキスト演出**
- 「逃避など許されない」のテキストが描画される。
- `escCnt` の値に応じてテキストが震える。

**フェード演出**
- 画面全体に半透明の白いフェードがかかる。
- フェードのアルファ値は `escCnt`の値に応じて変わる（ `escCnt × 8.5`（最大 255））。
- `escCnt = 30` で画面が完全に明転し、EscNightmare へ遷移する。

### EscNightmare シーン

- `hasEverEscapedNightmare = true` に設定される。
- 専用テキスト（「おかえりなさい。」）を 50 フレーム間表示する。
- 100 フレーム後に `Title` へ自動遷移する。
